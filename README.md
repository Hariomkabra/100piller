## 🏗️ 100Piller Construction Services Website
A responsive and modern web application showcasing comprehensive construction services, including Residential, Commercial, and Industrial offerings. The site features interactive service cards, detailed modal pop-ups, and a dynamic packages section, all optimized for user engagement.

## 🌐 Live Demo
Experience the live site here: https://100piller.netlify.app

## ✨ Features
Interactive Service Cards: Each service (Residential, Commercial, Industrial) includes an image, brief description, and a "Know More" button that triggers a modal with detailed information.

Dynamic Packages Section: Tabbed interface allowing users to switch between Residential and Commercial packages, each with detailed pricing and descriptions.

Responsive Design: Ensures optimal viewing experience across devices of various sizes.

Modern Aesthetics: Utilizes contemporary design principles with clean layouts and engaging visuals.
